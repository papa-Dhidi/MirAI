# MirAI
 MirAI merupakan website AI yang membantu pengguna mencari film yang mereka sendiri tidak mengetahui judulnya, dengan hanya mencari ciri-ciri filmnya MirAI dapat membantu menampilkan daftar film yang relevan. MirAI sendiri merupakan aplikasi web berbasis Django (Python Framework) dan dengan beberapa penggunaan API dari IMdb dan Gemini AI.
